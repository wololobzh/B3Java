== Consignes

Écrivez un programme pour déclarer deux tableaux, puis entrez les éléments du premier tableau de l'utilisateur et copiez tous ses éléments multpiés par deux dans le deuxième tableau

== Déroulement

Données d'entrée

Elément 1 : 3
Elément 2 : 7
Elément 3 : 4
Elément 4 : 9
Elément 5 : 8

Données de sortie
Les éléments du tableau source sont : 3, 7, 4, 9, 8
Les éléments du tableau dest sont : 3, 7, 4, 9, 8
